About

I use a speedtest.net Python API to capture metrics on my network performance throughout the day and store them in my personal MySQL database. This app provides visualizations of those metrics.

The speed tests are run on a Raspberry Pi which is involved in several other jobs that I have running at home. The machine is connected via ethernet through a powerline adapter, like so: 

| ------ |     | --------- |     | -------- |     | ----- |
| Router | <-> | Powerline | <-> | Ethernet | <-> | Raspi |
|        | <-> | Adapter   | <-> | Switch   | <-> |       |
| ------ |     | --------- |     | -------- |     | ----- |

That set up might diminish performance a little compared to a direct ethernet connection, but it should be faster than wifi. Besides, thats also how I hook up my personal computer so this should accurately reflect my network experience.
